# Siddharth Kumar Mishra Portfolio

A modern, professional portfolio website for Siddharth Kumar Mishra - Full-Stack Developer from Patna, Bihar.

## Features

- 🎨 Modern dark theme with cyan/neon blue accents
- 📱 Fully responsive (mobile-first design)
- ✨ Smooth animations with Framer Motion
- 🔗 GitHub API integration for projects
- 💫 Particle background effects
- 🖥️ Single-page navigation

## Tech Stack

- React 18 + Vite
- Tailwind CSS 3
- Framer Motion 11
- react-tsparticles
- Node.js (Backend)

## Getting Started

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

## Deployment

### Frontend (Vercel/Netlify)

1. Push code to GitHub
2. Connect repository to Vercel/Netlify
3. Build command: `npm run build`
4. Output directory: `dist`

### Backend (Render)

1. Create new Web Service on Render
2. Connect to GitHub repository
3. Start command: `npm start`
